# Portfolio

A clean, minimal portfolio/resume template designed for hackathon winners and builders.

## Setup

1. Open `index.html` in your browser to view the portfolio
2. Customize the content with your own information:
   - Name and tagline
   - Contact information
   - Hackathon wins and projects
   - Experience and skills
3. Deploy to any static hosting service (GitHub Pages, Netlify, Vercel, etc.)

## Customization

The template uses CSS variables for easy theming. Modify the `:root` section in the `<style>` tag to change colors:

- `--bg`: Background color
- `--paper`: Card background
- `--ink`: Primary text color
- `--acc`: Accent color (blue)
- `--gold`: Award badge color

## Features

- Responsive design
- Print-friendly
- No external dependencies
- Fast loading
- Clean, professional aesthetic
